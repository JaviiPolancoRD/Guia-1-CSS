Guía Básica de CSS

1. Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para describir la presentación de un documento escrito en HTML o XML. Permite controlar el estilo visual de las páginas web, incluyendo colores, fuentes, márgenes, entre otros. CSS ayuda a separar la presentación del contenido, facilitando la gestión y el mantenimiento de los estilos.

2. Funcionamiento Básico de CSS

CSS funciona a través de reglas que aplican estilos a los elementos de un documento HTML. Los navegadores interpretan estas reglas para mostrar el documento. CSS usa un sistema de cascada, herencia y especificidad para controlar cómo se aplican los estilos en caso de conflictos.

3. Incluir CSS en un Archivo HTML

  • CSS Externo: Vinculando un archivo CSS externo con la etiqueta <link>: 
    <link rel="stylesheet" href="estilos.css">
  • CSS Interno: Escribiendo el CSS en una etiqueta <style> en el <head>: 
    <style>
    body {
        background-color: #f0f0f0;
    }
</style>
  • CSS en Línea: Agregando estilos directamente en los elementos HTML usando el atributo style: 
    <p style="color: red;">Texto en rojo.</p>

4. Definición de una Regla CSS

Una regla CSS es un bloque de código que especifica los estilos a aplicar a elementos HTML. Cada regla consta de un selector y un bloque de declaración.

5. Elementos de una Regla CSS

  • Regla: La combinación de un selector y un bloque de declaración.
  • Selector: Especifica a qué elementos HTML se aplican los estilos. Ejemplo: p, .clase, #id.
  • Bloque de Declaración: Contiene una o más declaraciones de propiedades y valores, rodeadas por llaves { }.
  • Propiedad: Define el aspecto que se modificará (como color, font-size, margin).
  • Valor: Especifica el valor asignado a la propiedad (por ejemplo, red para el color).

Ejemplo de una regla CSS completa:

p {
    color: blue;
    font-size: 14px;
}

6. Medios CSS

CSS permite aplicar estilos específicos según el dispositivo mediante las media queries, que ayudan a crear sitios web responsivos. Ejemplo:

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

7. Comentarios CSS

Los comentarios en CSS se utilizan para añadir notas explicativas sin afectar el código. Se escriben entre /* y */:

/* Este es un comentario */
body {
    background-color: white;
}

8. Sintaxis de la Definición de las Propiedades CSS

La sintaxis de una propiedad CSS sigue el formato propiedad: valor;. Cada declaración se separa por un punto y coma ;. Ejemplo:

h1 {
    font-size: 24px;
    color: #333;
    margin: 10px 0;
}
